# 参与贡献

欢迎通过文档、设计、测试与协议讨论加入 VibeAgent 生态。

## 仓库可见性

| 仓库 | 可见性 | 外部贡献 |
|------|--------|----------|
| [docs](https://github.com/AgentSkillMesh/docs) | **公开** | ✅ Fork / PR / Issue |
| [contracts](https://github.com/AgentSkillMesh/contracts) | 私有 | 待开源 |
| [api](https://github.com/AgentSkillMesh/api) | 私有 | 待开源 |
| [web](https://github.com/AgentSkillMesh/web) | 私有 | 待开源 |
| p2p / shared | 私有 | 待开源 |
| [VibeAgent](https://github.com/AgentSkillMesh/VibeAgent) MetaRepo | 私有 | 团队内 |

产品可供用户使用稳定后，将按模块逐步开放代码仓库。

## 当前欢迎的贡献

- 改进 Rspress 文档（用户/开发者/愿景/白皮书）  
- 翻译（中/英）  
- 协议改进提案（通过 docs Issues 讨论）  
- 安全负责任披露（见 [开源与安全](/developers/open-source-security)）  

## 文档贡献流程

1. Fork [AgentSkillMesh/docs](https://github.com/AgentSkillMesh/docs)  
2. 分支 `feature/xxx` 或 `fix/xxx`  
3. 本地 `pnpm dev` 预览  
4. 发起 PR，描述清晰并关联 Issue  

## 行为准则

尊重、包容、对事不对人。恶意行为将被拒绝。

## 本地文档站

```bash
git clone https://github.com/AgentSkillMesh/docs.git
cd docs
pnpm install --ignore-scripts
pnpm dev
```
