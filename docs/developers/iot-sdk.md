# IoT SDK · Bring Your Own Device

> **状态**：v0.4 规划。当前可先阅读规格 [IOT.md](/technical/IOT)。

## 定位

轻资产、重软件：设备厂商或开发者 **自带硬件**，通过 SDK 完成：

- 链上 **Device ID** 注册  
- **稳定币收款** 与订阅计费  
- 遥测数据 **attestation** 上链  
- 与 **Agent** 发现/报价/签单对接  

## 认证等级

| 等级 | 说明 |
|------|------|
| L0 | SDK 自助注册 + 质押防 spam |
| L1 | 厂商型号白名单 + 固件签名 |
| L2 | 企业资质（充电桩、冷链等） |

## 目标场景（按版本）

| 版本 | 示例 |
|------|------|
| v0.4 | 充电桩收款；车载 Agent 支付 |
| v0.5 | 气象传感器按秒卖数据 |
| v0.6 | 光伏余电报价；冷链温湿度 Oracle |
