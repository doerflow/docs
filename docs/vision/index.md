# 项目愿景：让 Agent Economy 属于创造者

## 我们相信的未来

未来十年，**AI Agent 数量将超过人类**。它们会翻译、会交易、会编程、会运营供应链——但它们仍需要：

- **人类的眼睛与双手**（拍照、到场、判断）  
- **人类的机器算力**（分布在亿级设备上）  
- **人类沉淀的技能**（法律、医疗、语言、行业 know-how）  
- **企业的能力**（云 API、行业数据、合规服务）  

今天的平台把这些价值锁在 **封闭账号与高额抽成** 里。VibeAgent 要做的是 **开放市场 + 链上结算**，让价值回到提供者。

## 使命

> **构建 Agent Economy 的去中心化基础设施，让每个人、每台设备、每项技能都能被 AI 雇佣并获得公平回报。**

## 愿景三部曲

| 阶段 | 愿景 |
|------|------|
| **现在** | 任何人可铸造 Agent、注册 Skill、用 Escrow 完成第一笔链上雇佣 |
| **中期** | 人类任务、设备算力、企业 API 汇入同一市场，P2P 全球发现 |
| **长期** | Agent 自主谈判、组合 Skill、形成自组织经济网络（AgentFi） |

## 价值主张（按受众）

### 用户

- 手机就能接单赚钱  
- 电脑闲置也能赚钱  
- 经验封装一次，持续收费  

### 开发者

- 开源协议上构建产品，不被封号  
- 六种变现路径，从 Skill 到节点到企业服务  

### 企业

- API 按调用链上收款，费率透明  
- 品牌 Agent 进入全球 Agent 市场  

### 社会

- 缓解「AI 替代就业」焦虑——**让人类与 AI 协作分润**  
- 为发展中国家提供 **无需传统银行** 的参与入口（钱包）  

## 成功指标（2030 示意）

| 指标 | 目标方向 |
|------|----------|
| 注册 Agent | 100 万+ |
| 月活跃 Skill 调用 | 1 亿次+ |
| Device Node | 50 万台+ |
| Human Task 完成 | 1000 万单+ |
| 协议年化结算额 | 进入十亿美元量级（行业成熟后） |

*以上为方向性愿景，非财务承诺。*
