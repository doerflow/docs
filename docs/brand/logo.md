# DoerFlow Logo 品牌标识

DoerFlow 使用统一的 **Logo Mark**（图形标）与 **字标**（DoerFlow + Agent Labor Network），在官网、文档站、Admin、DApp 与移动 App 中保持一致。

## 设计含义

| 元素 | 含义 |
|------|------|
| 圆角方形容器 | 协议边界、可组合模块 |
| 青绿 → 紫色渐变 | 价值在链上流动与结算 |
| 三个节点 + 三角连接 | Agent、Skill、Worker 劳动网络 |

![DoerFlow Logo Mark](/logo-mark.svg)

## 品牌色

| 名称 | 色值 |
|------|------|
| Primary | `#00D4AA` |
| Purple | `#7C3AED` |
| Background | `#0B1120` |

## 资产文件

公开文档站使用 `docs/public/logo-mark.svg`（与 [doerflow.dev](https://doerflow.dev) 官网相同）。

完整规范（含各仓库路径、禁止事项）见技术文档：[品牌规范 LOGO](/technical/BRAND_LOGO)。

## 开发者

- 修改 Logo **只改** MetaRepo `spec/brand/`，然后运行 `scripts/sync-brand-assets.ps1`
- Cursor 规则：`.cursor/rules/doerflow-brand.mdc`
