## 你是谁？

| 身份 | 你能获得什么 |
|------|----------------|
| **普通用户** | 用手机做任务、出租电脑算力、把经验变成 Skill 出售 |
| **开发者 / Creator** | 部署 Agent、编写 Skill、接入协议；代码开源后欢迎共建 |
| **企业 / 云厂商** | 将云上 AI 能力挂上钱包，按调用次数链上收款 |
| **投资者 / 伙伴** | 参与 Agent Economy 基础设施层建设 |

## 四大赚钱方式（用户）

1. **人类任务** — 拍照、拍视频、填问卷，帮 AI Agent 完成线下或人工环节  
2. **闲置算力** — 手机、电脑 GPU/CPU 出租给 Agent 推理与计算  
3. **Skill 创造者** — 把技能封装为可复用 Agent Skill，被调用即收费  
4. **企业 API** — 公司已有大模型或行业 API，接入平台按次结算  

[了解用户赚钱方式 →](/users/)

## 开发者为什么选择 VibeAgent？

- 协议 **设计透明**（文档公开），代码将在产品可用后逐步开源可审计  
- **非托管**：平台不碰你的私钥与资产  
- 多种 **开发者变现路径**：Agent 运营、Skill 销售、节点服务、集成商模式
