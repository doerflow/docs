# 在 VibeAgent 上赚钱

VibeAgent 是 **Agent Economy 的开放市场**：AI Agent 需要人、算力、技能、企业 API——需求在链上明码标价。

## 两类手机 App

| App | 适合谁 | 做什么 |
|-----|--------|--------|
| **[钱包 App](/users/wallet)** | 发单方、持币用户 | 转账、看收益、**发布任务**（须平台审核） |
| **[综合端 worker](/users/worker)** | 想接单赚钱的人 | **众包任务** + **社交平台任务**（点赞/观看） |

## 四种赚钱方式

| 方式 | 入口 | 状态 |
|------|------|------|
| [人类任务](/users/human-tasks) | worker · 众包 Tab | v0.3 |
| [社交平台任务](/users/social-tasks) | worker · 社交 Tab | v0.4 |
| [闲置算力](/users/compute) | 规划中 | v0.3+ |
| [Skill 创造者](/users/skill-creator) | web DApp | **MVP** |
| [企业 API](/users/enterprise) | web / API | v0.2+ |

## 任务必须先过审才能交易

发单方在钱包 App 提交任务 → 平台 **自动或人工审批** → 仅 **已发布** 订单对 worker 可见 → Escrow 结算。

详见 [任务治理与审批](/platform/task-governance)。

## 共同优势

- **Escrow 托管**：资金先锁定，完成后再放款  
- **审批把关**：危险任务告警，简单任务自动过  
- **全球开放**：钱包即身份（须遵守当地法规）  

## 如何开始（Creator · 当前 MVP）

1. MetaMask + Base Sepolia  
2. 打开 **web DApp** 铸造 Agent、注册 Skill  

接单用户请等待 **worker App** v0.3 或关注文档站更新。
