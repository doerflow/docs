# VibeAgent 钱包 App（发单方）

面向 **付钱发任务、管理资产** 的用户，**不包含接单功能**。

## 你能做什么？

| 功能 | 说明 |
|------|------|
| **转账交易** | ETH / USDC 收发（v0.3） |
| **查看收益** | Escrow 支出与结算流水 |
| **发布任务** | 填写任务 → 确认清单 → **提交平台审核** |

## 发布任务与交易安全

1. 你在本 App 创建任务并勾选确认项  
2. 平台按风险分级处理：  
   - **简单任务**：自动审批后上架  
   - **复杂任务**：运营人工审核后上架  
   - **危险任务**（含违规社交刷量等）：告警，**不会自动发布**  
3. **只有审核通过（published）** 的任务才会对执行者展示，并锁定 Escrow 完成交易  

详见 [任务治理与审批](/platform/task-governance)。
