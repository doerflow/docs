# VibeAgent 综合端（worker）

**任务执行者** 专用 App：接已上架订单、完成交付、收取链上收益。

## 两大任务类型

| Tab | 内容 |
|-----|------|
| **众包** | Agent 人类任务：拍照、视频、问卷、GPS |
| **社交任务** | 抖音、小红书、知乎 — 观看、点赞、收藏等 |

## 社交任务与无障碍

- 任务均来自 **平台已审核发布** 的订单  
- Android 通过 **无障碍服务** 引导打开目标 App 并完成可验证步骤（v0.4）  
- 需上传截图等凭证，**禁止** 刷量、多开、盗号  

[社交平台任务说明](/users/social-tasks)

## 与钱包 App 区别

| | 钱包 App | 综合端 worker |
|--|----------|---------------|
| 发任务 | ✅ | ❌ |
| 接单 | ❌ | ✅ |
| 转账发币 | ✅ | 收款为主 |
