# 社交平台任务：点赞、观看、赚钱

在 **综合端 worker** 中，你可接发单方通过平台审核的 **社交类任务**，在自有账号上完成指定操作并获得 Escrow 报酬。

## 支持平台（规划）

| 平台 | 示例操作 | 版本 |
|------|----------|------|
| 抖音 | 观看视频、点赞 | v0.4 |
| 小红书 | 浏览笔记、点赞、收藏 | v0.4 |
| 知乎 | 阅读、赞同 | v0.5 |

## 如何完成？

1. 在 worker App「社交任务」Tab 选择 **已上架** 订单  
2. 阅读步骤说明（打开哪个 App、停留多久、点什么）  
3. （v0.4）开启 **无障碍辅助** 按引导操作 — 仅辅助，不代替你决策  
4. 截图上传 → 平台验证 → Escrow 放款到你的收款地址  

## 合规与风险

- 你必须使用 **自己的账号**，自愿操作  
- 平台 **禁止** 自动化批量注册、破解、窃取数据  
- 社交类任务默认 **人工+风控审批**，不走自动发布  
- 请遵守当地法律与目标平台用户协议  

## 发单方注意

在 **钱包 App** 发布社交类任务时，需如实填写平台与步骤；虚假或刷量类描述将触发 **L3 告警** 并被拦截。
