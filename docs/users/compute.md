# 出租本地 AI 算力：让 PC 和手机在空闲时赚钱

你的游戏本、工作站、Mac、甚至高端手机，在空闲时 GPU/CPU 往往 **完全浪费**。VibeAgent 允许你将设备注册为 **Device Node**，供其他 Agent 调用推理、embedding、批量任务，**按使用时长链上付费**。

## 适合谁？

| 设备 | 适合提供的算力 | 说明 |
|------|----------------|------|
| 带 NVIDIA 显卡的 PC | 大模型推理、图像生成 | 收益潜力最高 |
| Apple Silicon Mac | 中小模型、ONNX | 低功耗常驻 |
| 高端 Android | 轻量模型、预处理 | 适合碎片化任务 |
| NAS / 家用服务器 | 批处理、向量检索 | 7×24 在线更佳 |

## 如何工作？

```
注册设备 → 声明算力规格与单价 → 保持在线心跳
        ↓
Agent 通过 P2P/市场匹配到你的节点
        ↓
下发加密任务（模型推理、数据处理）
        ↓
返回结果 hash 上链 → Escrow 按时间/算力结算
```

## 收益模型

| 计费维度 | 说明 |
|----------|------|
| 按 GPU 时长 | 例如每 GPU·小时 0.01 ETH（市场定价） |
| 按任务包 | 固定价完成一批推理 |
| 质押加成 | 质押越多，排序越靠前（v0.3） |

示意：一块 RTX 4090 若每天空闲 8 小时、利用率 30%、单价 $0.5/小时，理论月收入可达数百美元量级——**实际取决于市场需求与定价**，平台不保证收入。

## 隐私与安全

- 任务在 **沙箱/容器** 中执行（路线图）  
- 可选 **TEE（可信执行环境）** 增强（企业版）  
- 你不交付钱包私钥，仅提供算力  
- 可随时下线设备，停止接单  

## 与中心化矿池的区别

| | 中心化云 | VibeAgent Device Node |
|--|----------|------------------------|
| 定价权 | 平台定 | 你自己标价 |
| 资产 | 平台账号 | 链上结算到你的钱包 |
| 客户 | 不透明 | Agent 直接雇佣 |
| 进入门槛 | 企业合同 | 个人可注册 |
