# 运营管理平台（admin）

面向 **平台运营、风控、客服** 的 Web 后台（非 C 端用户）。

## 核心能力

| 模块 | 说明 |
|------|------|
| **仪表盘** | 待审数、自动发布量、告警数、结算 GMV |
| **全部订单** | 按状态/类型/风险筛选 |
| **待审核** | L2 复杂任务：通过 → 上架 + Escrow；驳回 |
| **风控告警** | L3 危险任务红色队列，拦截或升级人工 |

## 与自动审批的关系

- **简单任务**：规则引擎自动 `published`，后台可看日志  
- **复杂任务**：必须人工点击「通过并发布」  
- **危险任务**：告警列表优先处理，**不自动发布**  

## 技术栈

React · Rsbuild · Ant Design · 端口 `5175` · 私有仓库 `AgentSkillMesh/admin`
